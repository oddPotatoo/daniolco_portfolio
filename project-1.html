OCR Access Control System App

An Optical Character Recognition based access control system developed for gated areas to extract needed information from valid IDs in the Philippines.

Code Link

Project Overview

Current manual ID checks at subdivision gates, private offices, gated areas, cause 1-5-minute delays per individual, leading to inefficiency, especially during peak hours. Security personnel face operational challenges due to legal restrictions on retaining IDs and inefficient logbook processes.

As a solution, the creation of automated system leveraging Optical Character Recognition (OCR) to scan valid IDs, extract non-sensitive data (such as name, ID type, last 4 digits of ID number), and generate QR codes for temporary access passes can streamline the entrance and exit of visitors and provide a more reliable system compared to manual methods. This app is integrated with a web monitoring app to see the entry and exit logs of visitors.

Key Benefits:

  • Reduced wait times: Target processing time of <10 seconds per vehicle.
  • Legal compliance: Eliminates manual ID retention by security personnel.
  • Traffic flow improvement: Streamlined entry process minimizes congestion.
  • Enhanced security: Encrypted data storage and audit trails via cloud synchronization.

My Role

As part of the development team for the OCR Access Control System App, I took the lead in designing the overall system architecture and defining the process flows for each module to ensure smooth and logical operations. I was responsible for the seamless integration of hardware and software components, enabling real-time access control functionality. Utilizing Python, I programmed the core application and crafted a user-friendly interface to enhance usability. I also lead the designing and integration of Firebase Real-time database and PostgreSQL for the app. Additionally, I conducted thorough system testing in conjunction with the hardware components to validate performance, accuracy, and reliability.
